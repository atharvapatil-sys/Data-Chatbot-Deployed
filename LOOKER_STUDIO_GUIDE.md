# Looker Studio Integration Guide

InsightStream allows you to seamlessly explore your data in Google Looker Studio. You can do this in two ways:

## 1. Quick Export (Built-in)

Every time Gemini generates a chart, you will see a **Looker Studio icon** (next to the CSV Download button).
- Clicking this will open Looker Studio's "Quick Report" creator.
- It is pre-configured to use the BigQuery connector.
- You can immediately start dragging and dropping fields from the `ulipsu-ai` project into a professional dashboard.

## 2. Permanent Dashboard Connection

If you want to build a long-term dashboard that refreshes automatically:

1. Go to [Looker Studio](https://lookerstudio.google.com/).
2. Click **Create** > **Data Source**.
3. Select the **BigQuery** connector.
4. Select your Project: `ulipsu-ai`.
5. Select the Dataset and Table (e.g., `marketing_spend`).
6. Click **Connect**.
7. Now you can use these tables as the data source for any Looker Studio Report.

## 3. Custom Query (Advanced)

If you have a complex SQL query generated by InsightStream that you want to visualize:
1. In Looker Studio, select **BigQuery** > **Custom Query**.
2. Select the `ulipsu-ai` project.
3. Paste the SQL generated by InsightStream into the query box.
4. Looker Studio will now treat that query result as a live data source.

---

**Note**: Ensure your Google account has `BigQuery Data Viewer` permissions on the `ulipsu-ai` project to see the data in Looker Studio.
